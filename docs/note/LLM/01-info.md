#### 在 LLM 学习中的应用

- **记忆**：了解 LLM 的基本概念和术语
- **理解**：理解 Transformer 架构的工作原理
- **应用**：能够使用 LLM API 解决实际问题
- **分析**：分析不同模型架构的优缺点
- **评价**：评估模型输出的质量和可靠性
- **创造**：设计新的模型架构或应用场景
